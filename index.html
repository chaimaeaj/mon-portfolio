Mes Travaux

TP1 - Base de données

Description : Conception et optimisation d'une base de données relationnelle pour une bibliothèque. J'ai modélisé les entités, rédigé des requêtes SQL avancées et amélioré les performances de la base pour une gestion fluide.

TP2 - Réseaux

Description : Mise en place d'un réseau local d'entreprise sécurisé. Ce projet a consisté à définir la topologie du réseau, configurer les routeurs et mettre en œuvre des protocoles de sécurité pour garantir l'intégrité et la confidentialité des données.

TP3 - Développement Web

Description : Développement d'une application web dynamique de A à Z. J'ai géré le frontend avec HTML/CSS/JS et le backend avec PHP, tout en connectant l'application à une base de données MySQL pour une expérience utilisateur complète.

TP4 - Sécurité

Description : Réalisation d'un audit de sécurité complet sur une application web. J'ai effectué des tests de pénétration et analysé les vulnérabilités pour proposer des recommandations précises et renforcer la protection contre les menaces.
